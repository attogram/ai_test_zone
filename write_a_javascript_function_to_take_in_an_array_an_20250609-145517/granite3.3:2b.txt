Here's a JavaScript function that converts an array into an HTML unordered list (`<ul>`) based on the provided array elements:

```javascript
function arrayToList(arr) {
  let listHtml = '<ul>';

  for (let i = 0; i < arr.length; i++) {
    listHtml += `<li>${arr[i]}</li>`;
  }

  listHtml += '</ul>';
  return listHtml;
}
```

This function, `arrayToList`, takes in an array (`arr`) as its parameter and returns a string containing the HTML representation of the array elements as a list. Each array element is wrapped inside `<li>` tags, and all `<li>` elements are collected within a single `<ul>` tag.

Here's how you can use this function:

```javascript
const items = ['Apple', 'Banana', 'Cherry'];
const htmlList = arrayToList(items);
console.log(htmlList);
// Output: <ul><li>Apple</li><li>Banana</li><li>Cherry</li></ul>
```

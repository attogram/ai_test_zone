Here's a JavaScript function that takes in an array and outputs an HTML unordered list:

```javascript
function createList(arr) {
  var ul = document.createElement('ul');
  for (var i=0; i<arr.length; i++) {
    var li = document.createElement('li');
    li.appendChild(document.createTextNode(arr[i]));
    ul.appendChild(li);
  }
  return ul;
}
```

To use this function, you can pass an array of items to it and call the `createList()` function:

```javascript
var myListArray = ['Apple', 'Banana', 'Cherry'];

// Call createList() with the array as parameter
var myHtmlList = createList(myListArray);

// Append the HTML list to a parent element in your document, for example the body
document.body.appendChild(myHtmlList);
```

This will generate an unordered list of the items in `myListArray` and append it to the body of the HTML document.

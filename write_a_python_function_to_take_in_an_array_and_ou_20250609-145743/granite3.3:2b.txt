Here is a Python function that converts a list into an HTML unordered list (`<ul>`) based on your request:

```python
def list_to_html(input_list):
    html_list = "<ul>\n"
    for item in input_list:
        html_list += "  <li>" + str(item) + "</li>\n"
    html_list = html_list.rstrip() + "\n</ul>"
    return html_list
```

This function takes a Python list as input and returns a string containing an HTML unordered list (`<ul>`) with each element of the input list as a list item (`<li>`). The resulting HTML string can be used in a web page.

You can use this function like so:

```python
input_list = [1, 2, 3, 4, 5]
html_output = list_to_html(input_list)
print(html_output)
```

This will output the following HTML string:

```html
<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
  <li>5</li>
</ul>
```

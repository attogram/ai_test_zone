Here's a Python function that takes in a list (array) and returns a string containing an HTML unordered list of the array contents:
```python
def array_to_html_list(arr):
    """
    Takes in a list (array) and returns a string containing an HTML unordered list of the array contents.
    """
    html = "<ul>"
    for i, item in enumerate(arr):
        html += f"<li>Item {i+1}: {item}</li>"
    html += "</ul>"
    return html
```
You can use this function to convert an array to an HTML list. For example:
```python
arr = ["apple", "banana", "cherry"]
html_list = array_to_html_list(arr)
print(html_list)
```
This will output the following HTML code:
```html
<ul>
  <li>Item 1: apple</li>
  <li>Item 2: banana</li>
  <li>Item 3: cherry</li>
</ul>
```

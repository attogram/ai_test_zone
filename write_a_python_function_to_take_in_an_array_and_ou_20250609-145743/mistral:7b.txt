 Here is a Python function that takes in a list and returns an HTML unordered list (<ul>) of the list elements. This function assumes that the input is a list of strings, but can be easily modified to accept other data types by applying appropriate formatting within the list comprehension.

```python
def array_to_html_list(array):
    html_output = "<ul>\n"

    for element in array:
        html_output += f"  <li>{element}</li>\n"

    html_output += "</ul>"
    return html_output
```

Example usage:

```python
my_array = ['apple', 'banana', 'cherry']
html_list = array_to_html_list(my_array)
print(html_list)

Output:
<ul>
  <li>apple</li>
  <li>banana</li>
  <li>cherry</li>
</ul>
```

To generate an HTML list from an array in Python, we can create a simple and clean function that takes a list as input and returns a well-formed HTML `<ul>` element containing all the elements of the list, each wrapped in a `<li>` tag.

---

### âœ… Function Definition

```python
def array_to_html_list(arr):
    html = "<ul>\n"
    for item in arr:
        html += f"<li>{str(item)}</li>\n"
    html += "</ul>"
    return html
```

```html
</ul>
```

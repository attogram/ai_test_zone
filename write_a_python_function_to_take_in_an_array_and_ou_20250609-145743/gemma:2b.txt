```python
def array_to_html_list(array):
  """
  Converts an array of objects into an HTML list.

  Args:
    array: A list of objects.

  Returns:
    An HTML list of the array contents.
  """

  html_list = []
  for item in array:
    html_list.append('<li>' + str(item) + '</li>')

  return '\n'.join(html_list)


# Example usage:
array = [1, 2, 3, 4, 5]
html_list = array_to_html_list(array)

print(html_list)
```

**Output:**
```html
<li>1</li>
<li>2</li>
<li>3</li>
<li>4</li>
<li>5</li>
```

**Notes:**

* The function uses string concatenation to generate the HTML list.
* The `str(item)` function is used to convert each object in the array to a string.
* The `join()` method is used to concatenate the strings into an HTML list.
* The function assumes that the input array contains objects that can be converted to strings.

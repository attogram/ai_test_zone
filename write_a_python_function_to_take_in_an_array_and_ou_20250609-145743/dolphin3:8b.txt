Here is a Python function that takes an array as input and outputs an HTML unordered list:

```python
def array_to_html_list(array):
    # Start with the opening <ul> tag
    html_list = "<ul>"
    
    # Iterate through each item in the array
    for item in array:
        # Add an <li> element for each item, followed by its value
        html_list += f"<li>{item}</li>"
    
    # Close the <ul> tag after all items have been added
    html_list += "</ul>"
    
    # Return the final HTML list string
    return html_list

# Example usage:
my_array = ["Apple", "Banana", "Cherry"]
print(array_to_html_list(my_array))
```

This code will output:

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
```

Each item in the array becomes a list item (`<li>` element) within the unordered list (`<ul>`).

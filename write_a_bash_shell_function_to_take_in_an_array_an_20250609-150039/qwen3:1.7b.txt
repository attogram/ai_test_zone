To create a Bash shell function that takes an array and outputs an HTML list of its contents, you can follow the structure below. This function handles both empty arrays and arrays with elements containing spaces or special characters.

---

### ‚úÖ Function Definition

```bash
print-array-as-html() {
    local array=("$@")
    echo "<ul>"
    for element in "${array[@]}"; do
        echo "<li>$element</li>"
    done
    echo "</ul>"
}
```

---

### üîç How It Works

- **`local array=("$@")`**  
  This captures all the positional arguments passed to the function into an array. If the function is called with multiple arguments, they are stored in `array`.

- **`echo "<ul>"`**  
  Starts the `<ul>` (unordered list) tag.

- **`for element in "${array[@]}"`**  
  Loops through each element of the array.

- **`echo "<li>$element</li>"`**  
  Outputs a `<li>` (list item) tag with the element's value.

- **`echo "</ul>"`**  
  Ends the `<ul>` tag.

---

### üìå Example Usage

```bash

print-array-as-html "${arr[@]}"
```

This will output:

```html
<ul>
<li>apple</li>
<li>banana</li>
<li>cherry</li>
</ul>
```

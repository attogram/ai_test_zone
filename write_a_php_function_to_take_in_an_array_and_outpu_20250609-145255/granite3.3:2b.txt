Here's a PHP function that takes an array as input and outputs an HTML unordered list (`<ul>`) containing the array elements:

```php
function array_to_html_list($array) {
    $htmlList = "<ul>";

    // Iterate through the array using a foreach loop
    foreach ($array as $value) {
        $htmlList .= "<li>" . htmlspecialchars($value) . "</li>";
    }

    $htmlList .= "</ul>";

    return $htmlList;
}
```

You can use this function like so:

```php
$myArray = ["Apple", "Banana", "Cherry"];
echo array_to_html_list($myArray);
```

This will output:

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
```

Make sure to use `htmlspecialchars()` when echoing the list items to prevent any potential XSS vulnerabilities.

Sure! Here's a PHP function that takes in an array and outputs an HTML unordered list of the array contents:

```php
function generateList($array) {
  // Start the unordered list
  $html = '<ul>';

  // Loop through each element in the array
  foreach ($array as $item) {
    // Add a list item for each element
    $html .= '<li>' . htmlspecialchars($item) . '</li>';
  }

  // Close the unordered list
  $html .= '</ul>';

  return $html;
}
```

You can call this function by passing an array of items to it. It will generate and return the HTML code for the unordered list. Here's an example usage:

```php
$array = ['Apple', 'Banana', 'Orange'];
$htmlList = generateList($array);
echo $htmlList;
```

This will output the following HTML code:
```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>
```

Note: The `htmlspecialchars()` function is used to escape any special characters in the array items, which helps prevent potential security vulnerabilities like HTML injection.

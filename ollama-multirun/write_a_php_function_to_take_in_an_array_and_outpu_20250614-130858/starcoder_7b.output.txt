### Input Format:

### Output Format:
Print a formatted list with proper indentations.

    5 
    1 2 3 4 5
### Sample Output:
    * 
        1
        2
    * 
        3
        4
        5

### Explanation
The input list has 5 elements, so your program should output 5 lines of output with proper indentation.

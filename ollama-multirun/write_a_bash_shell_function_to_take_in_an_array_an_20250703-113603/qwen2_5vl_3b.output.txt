Certainly! Below is a Bash shell function that takes an array as input and outputs an HTML list of its contents.

```bash
#!/bin/bash

# Define the function
create_html_list() {
    # Check if the input is an array
    if [ "$#" -eq 0 ]; then
        echo "Usage: create_html_list <array>"
        return 1
    fi

    # Initialize an empty string to hold the HTML list
    html_list=""

    # Loop through the array elements
    for element in "$@"; do
        # Append each element to the HTML list with a bullet point
        html_list+="• $element\n"
    done

    # Output the HTML list
    echo "<ul>$html_list</ul>"
}

# Example usage
# Define an array
array=("Apple" "Banana" "Cherry")

# Call the function with the array
create_html_list "${array[@]}"
```

### Explanation:
1. **Function Definition**: The function `create_html_list` is defined to take an array as input.
2. **Input Validation**: The function checks if the number of arguments (`$#`) is zero. If it is, it prints a usage message and returns.
3. **HTML List Construction**: The function initializes an empty string `html_list` to build the HTML list.
4. **Loop Through Array**: The function loops through each element in the array and appends it to the `html_list` string, preceded by a bullet point (`•`).
5. **Output HTML List**: Finally, the function outputs the constructed HTML list using the `<ul>` and `<li>` tags.

### Example Output:
If you run the script with the array `["Apple" "Banana" "Cherry"]`, the output will be:

```html
<ul>
<li>Apple</li>
<li>Banana</li>
<li>Cherry</li>
</ul>
```

This HTML list is generated by the function and can be used in any HTML document.
